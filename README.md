# DocWhiz 📚

**DocWhiz** is a web-based project built using HTML, CSS, and JavaScript. It uses smart logic to help users simplify and understand technical documentation or code snippets by generating AI-powered explanations and examples.

## 🚀 Features

- 💡 Smart explanations from `script.js`  
- 📄 Clean and organized layout in `index.html`  
- 🎨 Beautiful UI designed with `style.css`  
- 🧠 Simulated AI behavior for demo and understanding  

## 📁 File Structure

docwhiz/
├── index.html # Main webpage
├── style.css # Styling for the UI
└── script.js # Core logic for AI explanations
